Bienvenido a la documentación de **Beztack**. Este proyecto es un monorepo SaaS moderno, diseñado para ser escalable, seguro y eficiente.

## Stack Tecnológico

El proyecto utiliza tecnologías de vanguardia para ofrecer la mejor experiencia de desarrollo y usuario:

- **Monorepo:** [Nx](https://nx.dev) para gestión eficiente del espacio de trabajo.
- **Frontend:** [React](https://react.dev) + [Vite](https://vitejs.dev) en `apps/ui`.
- **Backend:** [Nitro](https://nitro.unjs.io) en `apps/api`.
- **Landing:** Astro o similar en `apps/landing`.
- **Documentación:** [Fumadocs](https://fumadocs.dev) en `apps/docs`.
- **Autenticación:** [Better Auth](https://better-auth.com).
- **Pagos:** Integraciones con MercadoPago y Polar.

## Estructura del Proyecto

El repositorio está organizado en aplicaciones y paquetes compartidos:

- `apps/`: Aplicaciones desplegables (UI, API, Docs, Landing).
- `packages/`: Librerías compartidas y utilidades.

Explora las secciones de la barra lateral para profundizar en cada aspecto del desarrollo en Beztack.
